P5.JS Coded Dim Sum Drawstring Bag

Problem

I was given a class project called "Textile Design Project."


Objective

To create a code with an original pattern, using loops, custom functions with inputs, correct code indentations, and citations.

Skills Employed

  • Sewing
  • p5.JS
Figure 1. Sewn drawstring bag with a custom coded pattern.

Approach

Essentially:

  • I decided to make a pattern of types of dim sum: char siu bao, siu mai, and har gow.
  • I used Khan Academy to get coordinates of the custom (Bezier) curves.
  • Used p5.JS to implement my code and troubleshooted the code to create a repeatable pattern.
  • My teacher was able to get our custom pattern onto fabric.
  • I sewed the patterned fabric into a drawstring bag.

Figure 2. Finding bezier curves that would create the shape of each dim sum from Khan Academy

Figure 3. The coded dim sum pattern

Results

The code was succesfully coded and followed all requirements for the project. The pattern was printed onto a fabric which then was sewn into a drawstring bag!
